# Quality Control

Verify all production assets and deliverables meet quality standards.

## Guidelines

- Verify all chapter images were generated successfully before proceeding
- Confirm TTS audio files have non-zero duration and match chapter count
- Validate BGM track duration is sufficient for the full episode
- Check final video file exists, has expected duration, and correct resolution
- Report any missing or failed assets before marking production complete
