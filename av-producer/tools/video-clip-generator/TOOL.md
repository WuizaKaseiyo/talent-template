# Video Clip Generator

Transforms static illustrations into AI-animated video clips using PiAPI/Kling.

## Parameters

- **image_path** (string, required) — Path to source image
- **prompt** (string, required) — Motion/animation description
- **duration** (string, optional) — "5" or "10" seconds (default: "5")
- **output_path** (string, required) — File path to save video clip

## Cost

$0.13 per 5s clip, $0.26 per 10s clip
