# Scene Composition

Create detailed image prompts that produce consistent, high-quality AI illustrations.

## Guidelines

- Write image prompts detailed enough for AI image generation without additional context
- Maintain character appearance consistency across all chapters in an episode
- Define style reference once and apply uniformly throughout the episode
- Compose scene layouts with appropriate lighting, atmosphere, and background elements
- Match visual mood to the emotional tone of the narration
